User Meta Management

This handy plugin, with simple interface, allows you to manage the user meta information.

Features

  • Add, edit, or delete meta information of an individual user.
  • Easily search for users who have same meta key and value combination.
  • Omit meta information to be not shown by simply adding their keys to the excempted list.
  • Default meta keys cannot be deleted by the user.
